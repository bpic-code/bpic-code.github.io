# 📦 Installation

Voici comment installer et configurer le projet.

## Prérequis

- Node.js v14 ou supérieur
- npm ou yarn

## Étapes

### 1. Cloner le repository

```bash
git clone https://github.com/bpic-code/projet.git
cd projet
```

### 2. Installer les dépendances

```bash
npm install
```

### 3. Vérifier l'installation

```bash
npm --version
```
