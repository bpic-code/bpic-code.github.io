# 🚀 Démarrage Rapide

Soyez opérationnel en 5 minutes !

## Installation rapide

```bash
git clone https://github.com/bpic-code/projet.git
cd projet
npm install
npm start
```

## Premiers pas

1. Ouvrez `http://localhost:3000` dans votre navigateur
2. Connectez-vous avec vos identifiants
3. Explorez le tableau de bord
