# 📖 Glossaire

Dictionnaire des termes techniques utilisés dans le projet.

## A

**API**  
Application Programming Interface - Interface de programmation permettant la communication entre applications.

**Authentification**  
Processus de vérification de l'identité d'un utilisateur.

## D

**Database**  
Base de données - Système de stockage organisé des données.

## E

**Endpoint**  
Point d'accès d'une API HTTP.

## R

**Repository**  
Dépôt de code source - Lieu de stockage du code du projet.

**REST**  
Representational State Transfer - Style architectural pour les services web.

## U

**User**  
Utilisateur - Personne utilisant l'application.
