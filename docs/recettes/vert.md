# 🍝 Vert caca

Pour un beau vert bien caca

## Pots

- Dark Angel green
- ultramanrine blue


## Recette

### Étape 1
Mettre une croute de vert
### Étape 2
Remplir les creux en bleu jusqu'à lissage.
### Étape 3
Faire un gros caca
